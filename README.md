### DEPS
```bash
$ smlpkg sync
```
### COMPILE
```bash
$ mlton cli-arg-sml.mlb
```
### RUN (PLAY)
```bash
$ ./cli-arg-sml 
Usage = ./cli-arg-sml [min] [max]
$ ./cli-arg-sml 70
Usage = ./cli-arg-sml [min] [max]
$ ./cli-arg-sml 10 20
Random = 18
$ ./cli-arg-sml 10 30
Random = 27
$ ./cli-arg-sml 10 50
Random = 44
```
